Recomended SaaS set up

How to set up SaaS to work best?

Its best to follow this practice in order to make sure you get most out of SaaS

  • Set up Trial period
  • Create Plans
  • Set up or import prompts and determine to what plan tier each prompt belongs
  • Java Script
Setting up trial perioid and its interaction:

In this SaaS best output you will be able to get is by disabling Trial perioid in admin>settings>General settings under field where it says "Trial Period". We recommend setting it to 0 because if you set trial perioid to ==>1+ it will allow your client to see any prompt tier and its content beafore he pays. In order to manage and controll free trial its best to set free prompt tier and trought it allow use of AI tools. In such case user will see only this tier if he signs up for it as free user and others will be locked. Please not if you set this tier free in billing settings it will allow copy of AI prompt content. If you need to enable trial perioid, in order to allow user promt discovery please add trial tokens higher than 0. Do not add paid tokens as free as this wont work.

Setting up Plans:

When you are setting up plans its important to pre plan them and than create them. We mainly use plans in order to create many locked tabs, where user needs to upgrade to higher. Also its important to Known that all tiers will be locked untill they are purchased/paid. Setting up this tiers to prce less than 0.2 will make it open and cause all to be open. To avoid this do not set price to 0.

Creating and assigning prompts to specific plan tiers:

Now when we have couple plans created we can move to prompt creation. Here we wont talk about creating prompt but rather how to assign it to specific tier. For start please navigate to: admin>AI prompts>Prompt Templates and fill all requested data. When you have data ready, You will see field saying "Plan". Click on it and dropdown will autoload created plans and you should select one from list. When you are done with all data click save prompt. Note: If you do not create any prompt and have created plans your customer will see blank screen on tab click, so assigning prompts to specific plans is big priority in order to get SaaS fully functional.
