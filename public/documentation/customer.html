Customer Panel

Dashboard

User can create a new project using one of the pre-existing templates. They can also view all projects that they have created.

Create/Edit Project

If the user creates a project, they will be able to edit the project name, prompt the AI for content and save the content in the text-editor on the right side of the page. The text-editor gives them a space to create/proof read and finalize their content write-up.

Templates

User can select the AI template that they want to use for their new project.

Profile

User can edit their account information.

Billing

User can view billing details and select preferred plan.
