General Settings

After you've logged in to the admin area, go to settings page from the menu. On this page add your company name, date format, time format, currency, trial period (for new users) and upload your logo and favicon. Click Save once you are done.

Email Config

Under email config you have 2 options to setup; SMTP and email templates

  • SMTP Settings

    Obtain your email SMTP settings from your host and then add them here. Once you are done click save.

  • Email Templates

    ProAi Tools already has predefined email templates. To edit a template click edit and then change the entire email. You can click on the button labelled to setup HTML emails. Any item enclosed by {} is a variable and be careful not to remove it from the template.

Integrations
Open AI
  1. Go to (https://openai.com/api) and create or login to your Open AI account.
  2. On the top right click Personal and from the dropdown select View API Keys
  3. On the resulting page click create new secret. Copy the secret and save it in your script.
Stripe
  1. Sign up for a Stripe account: Go to the Stripe website (https://stripe.com/) and click on the "Sign Up" button in the top right corner of the page. Follow the instructions to create an account.
  2. Go to your Stripe Dashboard (https://dashboard.stripe.com/) and then click on the "Developers" button on the right-hand side of the page.
  3. Select API Keys from the left side menu. You'll see two keys: a publishable key and a secret key. Copy them to your stripe integration.
  4. To set up webhooks, select Webhooks from the left side menu. Click the "Add endpoint" button to add a new webhook endpoint. On the next screen, enter the following url https://yoururl/webhook/stripe (replace yoururl with your domain name). Select the All events to listen to and click Add Events. Click Add Endpoint. Copy the Signing secret to webhook secret field.

That's it! You now have your Stripe publishable key, secret key, and webhook secret, which you can use to integrate Stripe into your application. Make sure to keep your keys and secrets safe and secure to protect your payment information.

Google
  1. Go to the Google Cloud Console and create a new project. https://console.cloud.google.com/apis/dashboard then follow the screen below: As you can see there is a pop-up box. Just check the Terms of service then click Agree and Continue.
  2. Then click the CREATE PROJECT to continue. Then input your project name and organization (you can set it as No organization) then click CREATE.
    NOTE: If you have a project already you can skip the above steps. And just select your current project name created previously.
  3. After creating the project, click the credentials link on the left side menu. Then click + CREATE CREDENTIALS button. From the dropdown select OAuth client ID.
  4. In the next screen click CONFIGURE CONSENT SCREEN.
  5. In the next screen click, select User Type as External
  6. If your account is new you need to finish the OAuth consent screen and provide the important details they ask for.
  7. Once the consent is finished, go back to the Credentials page by clicking Credentials on the left menu. Then click + CREATE CREDENTIALS button & from the dropdown select OAuth client ID (Like step 3 above)
  8. In the next screen set the following:
    • Application type = Web application
    • Authorized Javascript origins = {Your url e.g. https://yoururl.com}
    • Redirect Uri = {https://yoururl.com/google/callback}
    and then click Create and copy your Client ID and Secret from the popup box

That's it! You now have your Google Client ID and Secret, which you can use to integrate Google into your application. Make sure to keep your keys and secrets safe and secure to protect your users' data.
